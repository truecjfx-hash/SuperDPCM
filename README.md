# SuperDPCM
力求把你气笑系列之气不笑

自用的奇奇怪怪音效格式之*SuperDPCM*

采样率是48000Hz，支持panning和volume调节。不限制声音长度。
普通的DMC采样可能播放会加快大概4/3倍。所以如果要正常播放采样，原本的声音文件需要进行一定的处理。
注意！原本的DPCM最高采样率约33144Hz。

所以怎么转换这个wav到DMC呢？
把一个正常采样放慢到0.69倍然后转DMC就行。
剩下该怎么播放，至少看一下那个.h文件即可。

一般来讲，把那个CJFX_Sound.c加入到你的工程，.h给include了就行——其实不加入那个工程的话可以链接那个.o。
那个.dev是干嘛的？Dev-C++工程文件。打开，编译，就是示例。
